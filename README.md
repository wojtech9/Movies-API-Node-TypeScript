Movies API

Run API :

1. npm install
2. Create .env file in root folder, which specified database's parameters
   .env structure :

   PORT=
   DB_TYPE=postgres
   DB_HOST=
   DB_PORT=
   DB_USERNAME=
   DB_PASSWORD=
   DB_NAME=

3. npm run dev

Test API:

1. npm run test

Build to production

1. npm run build
