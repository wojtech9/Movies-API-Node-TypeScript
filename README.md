Movies API

Run API :

1. npm install
2. Create .env file in root folder, which specified database's parameters
   .env structure :

   PORT= <br />
   DB_TYPE=postgres <br />
   DB_HOST= <br />
   DB_PORT= <br />
   DB_USERNAME= <br />
   DB_PASSWORD= <br />
   DB_NAME= <br />

3. npm run dev

Test API:

1. npm run test

Build to production

1. npm run build
